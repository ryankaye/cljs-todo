# ClojureScript Todo

A super simple Todo app built with ClojureScript and Reagent.

## Demo

[View demo](https://cljs-todo.vercel.app/)

## Start up

cd Documents/Clojure/todo/cljs-todo

npm start

## Build

npm run build
